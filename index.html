Termux merupakan aplikasi yang dapat digunakan untuk mengeksekusi sebuah script. Termux bisa diibaratkan sebuah terminal emulator layaknya seperti CMD, linux dan lain sebagainya. Namun sebelum menggunakan termux, pastikan kamu memiliki skill koding.

Namun apabila kamu masih pemula dan belum tahu menau perihal script atau bahkan koding, tidak perlu khawatir. Banyak berbagai script termux yang bisa digunakan untuk membantu segala macam aktivitas. Pahami beberapa perintah termux berikut ini:

  • kg install = perintah untuk menginstall package seperti python, php, bash dll.
  • pip install = perintah untuk menginstall mechanize, requests, bs4 dll.
  • git clone = perintah untuk mengclone script dari github.
  • cd = perintah untuk membuka folder/directory.
  • ls = perintah untuk menampilkan isi folder.
  • apt update = perintah untuk memperbarui program yang sudah terinstall.
  • apt upgrade = perintah untuk meningkatkan program ke versi yang lebih tinggi (opsional).

Perlu di ingat! kami tidak akan bertanggung jawab terhadap hal apa yang akan kamu lakukan terhadap script termux yang dibagikan dibawah ini. Admin hanya sekedar berbagi informasi dan wawasan selebihnya kamu sendiri yang bertanggung jawab!
